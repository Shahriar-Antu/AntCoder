# Installation

## Supported Platforms

| Platform | Method | Status |
|----------|--------|--------|
| Linux (x64/ARM64) | Binary / AppImage / Bun | ✅ |
| macOS (Intel/Apple Silicon) | Binary / Bun | ✅ |
| Windows (x64/ARM64) | Binary / NSIS Installer | ✅ |
| Web UI | Browser | ✅ |

## Method 1: Bun (Recommended)

```bash
# Install Bun if needed
curl -fsSL https://bun.sh/install | bash

# Clone and run
git clone https://github.com/Shahriar-Antu/AntCoder
cd AntCoder
bun install
bun run --cwd packages/opencode dev
```

## Method 2: Pre-built Binary

```bash
# Linux
curl -L https://github.com/Shahriar-Antu/AntCoder/releases/latest/download/antcoder-linux-x64 -o antcoder
chmod +x antcoder
./antcoder

# macOS
curl -L https://github.com/Shahriar-Antu/AntCoder/releases/latest/download/antcoder-darwin-arm64 -o antcoder
chmod +x antcoder
./antcoder

# Windows (PowerShell)
Invoke-WebRequest -Uri "https://github.com/Shahriar-Antu/AntCoder/releases/latest/download/antcoder-windows-x64.exe" -OutFile "antcoder.exe"
.\antcoder.exe
```

## Method 3: AppImage (Linux)

```bash
curl -L https://github.com/Shahriar-Antu/AntCoder/releases/latest/download/AntCoder-x86_64.AppImage -o AntCoder.AppImage
chmod +x AntCoder.AppImage
./AntCoder.AppImage
```

## Method 4: Windows Installer

Download `AntCoder-Setup-x64.exe` from [Releases](https://github.com/Shahriar-Antu/AntCoder/releases) and run it.

## Verify Installation

```bash
antcoder --version
# AntCoder 0.1.0

antcoder doctor
# ✅ Bun: 1.3.11
# ✅ llama-server: 0.1.0
# ✅ Model: qwen2.5-coder-3b (downloaded)
# ✅ Config: ~/.config/antcoder/antcoder.json
```

## Requirements

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| RAM | 2GB | 8GB+ |
| Disk | 3GB | 10GB+ |
| CPU | x64/ARM64 | AVX2+ / NEON |
| GPU | Optional | Metal (macOS), CUDA (NVIDIA), Vulkan (AMD/Intel) |

## GPU Acceleration

```bash
# Auto-detect (default)
antcoder --gpu-layers 99

# Force CPU only
antcoder --gpu-layers 0

# Metal (macOS)
antcoder --gpu-layers 99

# CUDA (NVIDIA)
antcoder --gpu-layers 99

# Vulkan (AMD/Intel)
antcoder --gpu-layers 99
```

## Troubleshooting

**Model download fails:**
```bash
# Set custom download directory
export ANTODER_MODELS_DIR=/path/to/models
antcoder

# Or pre-download
antcoder model download qwen2.5-coder-3b
```

**llama-server not found:**
```bash
# Install llama.cpp
brew install llama.cpp  # macOS
# or
sudo apt install llama.cpp  # Ubuntu

# Or let AntCoder use bundled version
export PATH="$PATH:$(bun pm cache dir)/llama.cpp/bin"
```

**Port 8080 in use:**
```bash
antcoder --port 8081
```
