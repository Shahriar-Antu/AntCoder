# Models

## Default Models

| Model | Size | RAM (CPU) | VRAM (GPU) | Context | Best For |
|-------|------|-----------|------------|---------|----------|
| **Qwen2.5-Coder-3B** | 1.85 GB | 3 GB | 2.2 GB | 32K | **Default** — General coding |
| Qwen2.5-Coder-1.5B | 1.0 GB | 2 GB | 1.5 GB | 32K | Fast completion, edge |
| Qwen2.5-Coder-0.5B | 0.4 GB | 1 GB | 0.8 GB | 32K | Minimal hardware |

All models: **Apache 2.0** license, quantized to **Q4_K_M** (4-bit).

## Switching Models

```bash
# CLI command
:model qwen2.5-coder-1.5b

# Config file
{
  "model": "local-llama/qwen2.5-coder-1.5b"
}

# Environment variable
ANTODER_MODEL=local-llama/qwen2.5-coder-1.5b antcoder
```

## Downloading Models

```bash
# Auto-download on first run (default)
antcoder

# Manual download
antcoder model download qwen2.5-coder-3b
antcoder model download qwen2.5-coder-1.5b

# List available
antcoder model list
```

## Custom Models

Add any GGUF model from HuggingFace:

```json
{
  "provider": {
    "local-llama": {
      "models": {
        "deepseek-coder-6.7b": {
          "name": "DeepSeek Coder 6.7B",
          "toolcall": true,
          "context": { "limit": 16384 }
        },
        "starcoder2-3b": {
          "name": "StarCoder2 3B",
          "toolcall": true,
          "context": { "limit": 16384 }
        }
      }
    }
  }
}
```

Then download:

```bash
antcoder model download deepseek-coder-6.7b
```

## Quantization Guide

| Suffix | Bits | Quality | Speed | Use Case |
|--------|------|---------|-------|----------|
| **Q4_K_M** | 4-bit | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | **Recommended** |
| Q5_K_M | 5-bit | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | Higher quality |
| Q8_0 | 8-bit | ⭐⭐⭐⭐⭐ | ⭐⭐ | Near FP16 |
| Q2_K | 2-bit | ⭐⭐ | ⭐⭐⭐⭐⭐ | Extreme compression |

## Model Storage

```
~/.config/antcoder/models/
├── qwen2.5-coder-3b-instruct-q4_k_m.gguf
├── qwen2.5-coder-1.5b-instruct-q4_k_m.gguf
└── qwen2.5-coder-0.5b-instruct-q4_k_m.gguf
```

Change location:

```json
{
  "llamaServer": {
    "modelsDir": "/custom/path/models"
  }
}
```
