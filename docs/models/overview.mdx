# Models

## Default Models

| Model | Size | RAM | Context | Best For |
|-------|------|-----|---------|----------|
| **Qwen2.5-Coder-3B** | 1.85 GB | 3 GB | 32K | **Default** — General coding |
| Qwen2.5-Coder-1.5B | 1.0 GB | 2 GB | 32K | Fast completion, edge devices |
| Qwen2.5-Coder-0.5B | 0.4 GB | 1 GB | 32K | Minimal hardware |

All models are **Apache 2.0** licensed.

## Switching Models

```bash
# CLI command
:model qwen2.5-coder-1.5b

# Config
{
  "model": "local-llama/qwen2.5-coder-1.5b"
}

# Environment
ANTODER_MODEL=local-llama/qwen2.5-coder-1.5b antcoder
```

## Downloading Models

```bash
# Auto-download on first run (default)
antcoder

# Manual download
antcoder model download qwen2.5-coder-3b
antcoder model download qwen2.5-coder-1.5b

# List available
antcoder model list
```

## Custom Models

Add any GGUF model from HuggingFace:

```json
{
  "provider": {
    "local-llama": {
      "models": {
        "deepseek-coder-6.7b": {
          "name": "DeepSeek Coder 6.7B",
          "toolcall": true,
          "context": { "limit": 16384 }
        },
        "starcoder2-3b": {
          "name": "StarCoder2 3B",
          "toolcall": true,
          "context": { "limit": 16384 }
        }
      }
    }
  }
}
```

Then download:

```bash
antcoder model download deepseek-coder-6.7b
```

## Model Storage

```
~/.config/antcoder/models/
├── qwen2.5-coder-3b-instruct-q4_k_m.gguf
├── qwen2.5-coder-1.5b-instruct-q4_k_m.gguf
└── qwen2.5-coder-0.5b-instruct-q4_k_m.gguf
```

Change location:

```json
{
  "llamaServer": {
    "modelsDir": "/custom/path/models"
  }
}
```
