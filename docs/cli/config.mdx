# Configuration Reference

## File Locations

| Platform | Path |
|----------|------|
| Linux | `~/.config/antcoder/antcoder.json` |
| macOS | `~/Library/Application Support/antcoder/antcoder.json` |
| Windows | `%APPDATA%\antcoder\antcoder.json` |

## Schema

```json
{
  "$schema": "https://antcoder.ai/config.json",
  "model": "local-llama/qwen2.5-coder-3b",
  "provider": { ... },
  "llamaServer": { ... },
  "agent": { ... },
  "theme": "catppuccin-macchiato",
  "keybindings": "default"
}
```

## Theme Options

| Theme | Description |
|-------|-------------|
| `catppuccin-macchiato` | Warm, colorful (default) |
| `catppuccin-frappe` | Light variant |
| `dracula` | Dark purple |
| `nord` | Arctic, bluish |
| `tokyonight` | Night blue |
| `github-dark` | GitHub style |
| `one-dark` | Atom One Dark |
| `monokai` | Classic Monokai |
| `solarized-dark` | Solarized |
| `gruvbox` | Retro groove |

## Keybinding Options

| Scheme | Description |
|--------|-------------|
| `default` | Standard bindings |
| `vim` | Vim-style |
| `emacs` | Emacs-style |
| `custom` | User-defined |

## Validation

```bash
# Validate config
antcoder config show

# Edit in $EDITOR
antcoder config edit

# Reset to defaults
antcoder config reset
```
