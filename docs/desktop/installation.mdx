# Desktop Installation

## Linux

### AppImage (Recommended)

```bash
# Download
wget https://github.com/Shahriar-Antu/AntCoder/releases/latest/download/AntCoder-linux-x64.AppImage

# Make executable
chmod +x AntCoder-linux-x64.AppImage

# Run
./AntCoder-linux-x64.AppImage
```

### Dependencies

| Distro | Command |
|--------|---------|
| Ubuntu/Debian | `sudo apt install libfuse2 libappindicator3-1` |
| Fedora | `sudo dnf install fuse-libs libappindicator-gtk3` |
| Arch | `sudo pacman -S fuse2 libappindicator-gtk3` |

### Desktop Entry (Optional)

```bash
# Create .desktop file
cat > ~/.local/share/applications/antcoder.desktop << 'EOF'
[Desktop Entry]
Name=AntCoder
Exec=/path/to/AntCoder.AppImage
Icon=antcoder
Type=Application
Categories=Development;IDE;
Terminal=false
EOF
```

## macOS

### DMG Installer

```bash
# Download
curl -L -o AntCoder.dmg https://github.com/Shahriar-Antu/AntCoder/releases/latest/download/AntCoder-darwin-arm64.dmg

# Mount and install
hdiutil attach AntCoder.dmg
cp -r /Volumes/AntCoder/AntCoder.app /Applications/
hdiutil detach /Volumes/AntCoder
```

### Gatekeeper

First run may show "App is from unidentified developer":

1. Right-click AntCoder.app → Open
2. Or: `xattr -d com.apple.quarantine /Applications/AntCoder.app`

## Verification

```bash

antcoder --version
# AntCoder 0.1.0

# Health check

# ✅ Bun: 1.3.11
# ✅ llama-server: 0.1.0
# ✅ Model: qwen2.5-coder-3b
# ✅ Config: ~/.config/antcoder/antcoder.json
```

## Uninstall

| Platform | Method |
|----------|--------|
| Linux | Delete AppImage + `~/.config/antcoder` |
| macOS | Drag AntCoder.app to Trash |
| Windows | Settings > Apps > AntCoder > Uninstall |

## Data Locations

| Platform | Config | Models | Logs |
|----------|--------|--------|------|
| Linux | `~/.config/antcoder` | `~/.config/antcoder/models` | `~/.local/share/antcoder/logs` |
| macOS | `~/Library/Application Support/antcoder` | Same | `~/Library/Logs/antcoder` |
| Windows | `%APPDATA%\antcoder` | Same | `%LOCALAPPDATA%\antcoder\logs` |
